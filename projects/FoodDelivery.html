Food Delivery Website

  • Tech Stack: HTML, CSS,JavaScript
  • Github URL: Project Link
  • Website URL: Project Link

Developed a web-based application for food delivery services, providing users with a seamless experience in ordering meals from various restaurants.

Implemented Pub/Sub functionality to handle real-time order updates and notifications, ensuring timely delivery to customers; utilized Dataflow for efficient data processing, optimizing order management and delivery logistics.

Conducted rigorous performance testing to evaluate system scalability and reliability under high user loads, simulating 10,000 concurrent orders to validate system robustness and stability.
